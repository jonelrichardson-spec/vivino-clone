A pixel-perfect clone of the Vivino wine app built to study the engineering behind interfaces that feel simple but aren't. The core challenge was replicating Vivino's two-layer scrollable architecture, where the outer page and inner wine list panel scroll independently.

Built during Pursuit's AI-Native Builder Fellowship (February 2026) with teammate Luba.

## What I Learned

Getting the scrollable layout right required careful attention to CSS overflow behavior, fixed vs. relative positioning, and how scroll events propagate through nested containers. Building this taught me how much design complexity hides inside interfaces that feel effortless to use.

- **Frontend**: React · Vite · Tailwind CSS
